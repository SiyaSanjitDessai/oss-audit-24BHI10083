## Project Description
This project is based on the study of open-source software. The Linux Kernel is selected as the software, and its origin, license, Linux footprint, and ecosystem are analyzed. Shell scripts are created to demonstrate Linux skills.

## Scripts Description

### Script 1: System Identity Report
Displays system information like distro, kernel, user, date, and uptime.

### Script 2: Package Inspector
Checks whether a package is installed and prints a philosophy note.

### Script 3: Disk and Permission Auditor
Displays directory size and permissions for key system directories.

### Script 4: Log File Analyzer
Counts lines containing a keyword in a log file and shows last 5 matches.

### Script 5: Open Source Manifesto Generator
Takes user input and creates a personalized open-source philosophy statement.

## How to Run Scripts
1. Give permission: chmod +x script.sh
2. Run script: ./script.sh

## Requirements
